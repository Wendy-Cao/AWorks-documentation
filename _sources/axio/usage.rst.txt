基本用法
========

查看帮助 ``-h , --help``
------------------------
 
axio 支持多层命令，每一层命令的帮助都可以通过加上 ``--help`` 或 ``-h`` 来查看，例如：

.. code-block:: bash

    $ axio --help
    Usage: axio [OPTIONS] COMMAND [ARGS]...

      axio is everything...

    Options:
      --version                    Show the version and exit.
      -d, --project-dir DIRECTORY
      -f, --force                  Force to accept any confirmation prompts.
      -h, --help                   Show this message and exit.

    Commands:
      build      buid targets...
      ci         Continuous Integration
      download   Download new packages
      init       Initialize axio project
      install    Install new packages
      list       List installed packages
      new        Begin new component development
      package    Package Manager
      repo       Repository Management
      settings   axio settings management
      uninstall  Uninstall packages
      update     Update installed packages
      upgrade    Upgrade axio to the latest version

.. code-block:: bash

    $ axio build --help
    Usage: axio build [OPTIONS]

    Options:
      -d, --project-dir DIRECTORY
      --upload-port TEXT
      -e, --environment TEXT
      -t, --target TEXT
      -s, --silent
      -v, --verbose
      -T, --target-repo TEXT
      --disable-auto-clean
      --disable-ci-check
      --force-register
      --with-vdrv                  useful to break path-length-limit of windows
      -h, --help                   Show this message and exit.

用户配置 ``axio settings``
--------------------------

.. code-block:: bash

    $ axio settings --help
    Usage: axio settings [OPTIONS] COMMAND [ARGS]...

    Options:
      -h, --help  Show this message and exit.

    Commands:
      get    Get existing setting/-s
      reset  Reset settings to default
      set    Set new value for the setting

.. code-block:: bash

    $ axio settings get
    Name                            Value [Default]   Description

    check_axio_interval             1                 Check for the new axio interval (days)
    disable_progressbar             Yes               Disable progressbar
    enable_cache                    Yes               Enable caching for API requests and package Manager
    force_verbose                   No                Force verbose output when processing environments
    repo_home                       http://share/public/docs/spd/artifacts/manifest.json  Repositories home url
    webdav_password                 ******** [password] Your password for http://share
    webdav_username                 robot-spd [uname] Your user name for http://share

对大部分用户而言只需要正确配置 ``webdav_username`` 和 ``webdav_password`` 即可：

.. code-block:: bash

    axio settings set webdav_username  your_pc_accout
    axio settings set webdav_password  your_pc_password

.. note:: 可打开 ``{$AXIO_HOME_DIR}/appstate.json`` 查看或修改配置，需要注意的是此文件只体现有修改的配置。

升级axio ``axio upgrade``
-------------------------

用户可以通过命令 ``axio upgrade`` 将本地的 axio 升级到最新版本。

.. code-block:: base
  
      $ axio upgrade
      You're up-to-date!
      axio 1.0.40 is currently the newest version available.

.. note:: axio 会自动检查更新，默认频率是每天检查一次，若发现有更新，会在每次执行 axio 命令后给出提示。用户可以修改用户配置的 ``check_axio_interval`` 来设置更新检查频率
