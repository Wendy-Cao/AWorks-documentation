3. AXIO 工具¶

AXIO 是下一代软件构建器，用于构建任意类型的软件制品，比如，文档、库、SDK等等。其特点如下：

  • 使用Python开发，构建器核心基于 scons ，支持 Windows 和 Linux [1]
  • 集成包管理器，可自动下载依赖的包，也可自动上传成功构建的包
  • 集成CI功能，可自动生成CI脚本
  • 扩展性强，可自定义 builder ， command 等等，且能够通过包管理器进行共享复用
[1]当前版本不支持
